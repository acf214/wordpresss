Chronicle is a responsive WordPress theme with multipurpose design. 

== Description ==
It is best suited for both small businesses and corporate businesses, as it is highly customizable via the Live Customizer. You can use Chronicle as restaurant theme, startups, freelancer, creative agencies, portfolios, WooCommerce, or niches like sports, medical, blogging, fashion, lawyer sites etc.

<a href="https://wordpress.org/themes/chronicle/">Discover more about the theme</a>.

== Frequently Asked Questions ==
